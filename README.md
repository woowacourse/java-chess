## 구현 기능

### 1. 체스판 초기화

- 체스판은 8x8 크기이다.
- 각 플레이어는 하나의 킹, 퀸과 2개의 비숍, 나이트, 룩, 8개의 폰을 가진다.
- 각 플레이어는 White 혹은 Black 진영을 가지고 시작한다.
- 체스판에서 말의 위치 값은 가로 위치는 왼쪽부터 a ~ h이고, 세로는 아래부터 위로 1 ~ 8로 구현한다.

### 2. 말 이동

#### 모든 기물

- 같은 팀 기물이 있는 곳으로 이동할 수 없다.
- 상대 팀 기물이 있는 곳으로 이동하면 상대 기물이 잡힌다.

#### 킹, 퀸, 룩, 비숍, 폰

- 이동 경로에 다른 기물이 있는 경우 뛰어 넘을 수 없다.

#### 니이트

- 이동 경로에 다른 기물이 있더라도 뛰어넘을 수 있다.

## 3. 승패 및 점수

- 상대편 King을 잡는 경우 게임에서 승리한다.
    - King이 잡혔을 때 게임은 종료된다.
- 현재 남아 있는 말에 대한 점수를 구한다.
    - Queen은 9점
    - Rook은 5점
    - Bishop은 3점
    - Knight는 2.5점
    - Pawn은 1점
        - 같은 세로줄에 같은 팀의 폰이 있는 경우 1점이 아닌 0.5점을 준다.

- 한 번에 한 쪽의 점수만을 계산해야 한다.
- 남은 기물의 점수의 합이 높은 쪽이 승리한다.

## 4. 게임 커맨드

- `start`: 게임을 시작하고 체스판을 초기화 한다.
- `end`: 게임을 종료한다.
- `move source위치 target위치`: source위치에 있는 기물을 target위치로 옮긴다.
- `status`: 각 진영의 점수를 출력하고 어느 진영이 이겼는지 결과를 보여준다.

## 6. 웹 게임 이용 방법

### 시작 화면

- localhost:4567 로 접속하면 웹으로 체스 게임을 즐길 수 있다.
- 체스 게임 이름을 정하고 체스 게임을 새로 시작할 수 있다.
    - 게임 이름은 빈칸일 수 없다.
    - 게임 이름은 중복될 수 없다.
- 게임 불러오기 기능을 통해서 이전에 했던 게임을 불러와서 게임을 이어서 플레이할 수 있다.
- 삭제 버튼을 통해서 진행중이던 이전의 게임을 삭제할 수 있다.

### 체스 게임 화면

- 체스 기물을 클릭하고, 움직이고 싶은 좌표를 선택해서 기물을 움직일 수 있다.
- 점수 확인 버튼을 통해서 현재 체스 판의 점수와 승자를 알 수 있다.
- 게임 저장 후 뒤로가기 버튼을 통해서 게임을 저장하고 시작 화면으로 돌아갈 수 있다.
- 게임 종료 버튼을 통해서 진행하던 게임을 종료하고 시작 화면으로 돌아간다.
