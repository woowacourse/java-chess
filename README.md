## 1단계 체스판 초기화

1. [x] 체스판은 8x8 사이즈이다.
2. [x] 체스의 말 종류는 P, R, N, B, Q, K 이렇게 6가지다.
3. [x] 진영은 대문자(검은색, 위쪽)와 소문자(흰색, 아래쪽)로 구분한다.
4. [x] `start`를 입력하면 체스판을 초기화한다.
5. [x] `end`를 입력하면 종료된다.


## 2단계 체스 말 이동
1. [ ] 룩은 직선으로 움직인다.
2. [ ] 비숍은 대각선으로 움직인다.
3. [ ] 퀸은 직선 + 대각선으로 움직인다.
4. [ ] 킹은 직선 + 대각선인데 이동 범위가 한칸으로 제한된다.
5. [ ] 나이트는 직선으로 한칸 이동하고 그 방향의 대각선으로 한칸 이동한다.
6. [ ] 폰은 처음 움직일 경우에는 직선 방향으로 한칸 또는 두칸 이동한다. 공격할 경우는 이동하는 방향의 대각선으로 한칸만 가능하다.
