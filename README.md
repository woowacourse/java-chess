##말 이동의 경우의 수

- 전제: 우리팀(흰색) 적팀(검은색)이 있다고 가정한다
3. 우리팀 -> 빈칸 (이동)
4. 빈칸 -> 빈칸 (명령어 실행 안됨)
5. 말들(룩, 퀸, 비숍, 폰)이 이동할 때 중간에 장애물이 있을 경우 이동시킬 수 없다
6. 말의 이동 규칙에 알맞지 않게 움직이면 이동 못한다
1. 우리팀 -> 적팀 (공격)
2. 우리팀 -> 우리팀 (이동 불가)
